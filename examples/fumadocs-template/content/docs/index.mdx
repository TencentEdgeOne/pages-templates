# Welcome to Fumadocs Template

This is a modern documentation site template built with [Fumadocs](https://fumadocs.dev), a powerful documentation framework powered by Next.js.

## Features

This template comes with everything you need to create professional documentation:

- **MDX Support** - Write documentation with Markdown and React components
- **Type-Safe** - Full TypeScript support with auto-generated types
- **Built-in Search** - Full-text search powered by Orama
- **Dark Mode** - Automatic dark mode support
- **Performance** - Optimized for speed with Next.js App Router
- **EdgeOne Ready** - Deploy seamlessly to Tencent EdgeOne Pages

## Quick Start

Get started in three simple steps:

### 1. Install Dependencies

```bash
pnpm install
```

### 2. Start Development Server

```bash
pnpm dev
```

### 3. Open Documentation

Visit [http://localhost:3000/docs](http://localhost:3000/docs) to view your documentation.

## Project Structure

```
fumadocs-template/
├── content/           # MDX documentation files
│   └── docs/         # Documentation content
├── app/              # Next.js App Router
│   ├── docs/        # Docs pages
│   └── api/         # API routes
├── lib/              # Utility libraries
└── source.config.ts  # MDX configuration
```

## What's Included

This template provides:

1. **Pre-configured Setup** - MDX processing, search, and theming ready to use
2. **Beautiful Components** - Callouts, tabs, steps, cards, and file trees
3. **Documentation Structure** - Organized sections for easy navigation
4. **Deployment Ready** - Optimized for EdgeOne Pages with edge caching
