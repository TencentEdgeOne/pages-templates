# Components

Fumadocs provides a rich set of built-in components for your documentation.

## Callout

Use callouts to highlight important information:

<Callout type="info">
  This is an informational callout.
</Callout>

<Callout type="warn">
  This is a warning callout.
</Callout>

<Callout type="error">
  This is an error callout.
</Callout>

### Usage

<Callout type="info">
  This is an informational callout.
</Callout>

<Callout type="warn">
  This is a warning callout.
</Callout>

<Callout type="error">
  This is an error callout.
</Callout>

### Props

| Prop | Type | Description |
|------|------|-------------|
| `type` | `'info' \| 'warn' \| 'error'` | Callout style variant |

## Code Block

Fenced code blocks with syntax highlighting:

```typescript
function greet(name: string): string {
  return `Hello, ${name}!`;
}

console.log(greet('World'));
```

With a title:

```typescript title="utils.ts"
export function formatDate(date: Date): string {
  return date.toLocaleDateString('en-US');
}
```

### Usage

Use triple backticks with language:

```typescript
function greet(name: string): string {
  return `Hello, ${name}!`;
}
```

```typescript title="utils.ts"
export function formatDate(date: Date): string {
  return date.toLocaleDateString('en-US');
}
```

</Tabs>

### Props

**Card:**
| Prop | Type | Description |
|------|------|-------------|
| `title` | `string` | Card title |
| `href` | `string` | Link URL (optional) |

| `external` | `boolean` | Whether link is external |

## File Tree

Display file structures:

<FileTree>
  <FileTree.Folder name="app" defaultOpen>
    <FileTree.File name="layout.tsx" />
    <FileTree.File name="page.tsx" />
    <FileTree.Folder name="docs">
      <FileTree.File name="layout.tsx" />
      <FileTree.File name="page.tsx" />
    </FileTree.Folder>
  </FileTree.Folder>
  <FileTree.File name="package.json" />
  <FileTree.File name="tsconfig.json" />
</FileTree>

### Usage

<FileTree>
  <FileTree.Folder name="app" defaultOpen>
    <FileTree.File name="layout.tsx" />
    <FileTree.File name="page.tsx" />
  </FileTree.Folder>
  <FileTree.File name="package.json" />
</FileTree>

### Props

**FileTree.Folder:**
| Prop | Type | Description |
|------|------|-------------|
| `name` | `string` | Folder name |
| `defaultOpen` | `boolean` | Open by default |

**FileTree.File:**
| Prop | Type | Description |
|------|------|-------------|
| `name` | `string` | File name |
