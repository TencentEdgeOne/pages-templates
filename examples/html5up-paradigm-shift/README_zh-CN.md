## 表单配置

此模板包含由 [Formspree](https://formspree.io) 驱动的动态联系表单。设置表单的步骤：

1. 在 [Formspree](https://formspree.io) 注册账户
2. 新建表单，复制表单 ID（如 `xabcdefg`）
3. 修改 `config.js` 文件并添加您的表单 ID：
   ```
   NEXT_PUBLIC_FORM:"xabcdefg"
   ```

表单现在将动态使用您的 Formspree 端点进行提交。
